Kimi Cap Recovery
ComputeGroupCap is an anti-dominance guard: a non-initial model group like Kimi can only count up to 75% of the previous epoch's total network weight. When Kimi tries to return above that limit, the excess is clipped until the network total catches up. “No e266 attack” rebuilds e266 from reconstructed entry commit weight, then carries that simulated total_weight forward into later cap checks.
No e266 Attack Reconstruction
e266 is rebuilt from participant/model commits at epoch entry: raw commit count × model scale factor. e267 keeps actual rows and only carries forward entry participant/model rows that did not appear in e267.
